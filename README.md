# ExxonMobil (XOM) Mid-Cycle Valuation

## Overview

This project values ExxonMobil (NYSE: XOM) using a Discounted Cash Flow (DCF) framework based on normalized mid-cycle operating performance.

Because energy companies are highly exposed to commodity price fluctuations, the analysis adjusts for extraordinary operating environments—including the 2020 COVID-19 downturn and the 2022 oil-price spike—to estimate ExxonMobil's sustainable long-term cash flow generating capacity.

The valuation focuses on determining ExxonMobil's intrinsic value under realistic mid-cycle industry conditions rather than relying on peak or trough earnings.

---

## Investment Question

**What is ExxonMobil worth under normalized mid-cycle operating assumptions?**

The analysis evaluates whether ExxonMobil's market valuation reflects sustainable operating performance or temporary commodity-driven conditions.

---

## Key Findings

### Mid-Cycle Normalization

To estimate sustainable profitability, the valuation excludes the abnormal operating environments of:

* 2020 (COVID-19 demand collapse)
* 2022 (oil-price spike)

This approach produces normalized assumptions for margins, tax rates, and reinvestment requirements.

### Reinvestment Analysis

Historical capital expenditures were adjusted to ensure consistency between projected growth and reinvestment needs. Rather than directly extrapolating historical capital spending, reinvestment was derived using the relationship between growth and return on invested capital (ROIC).

### Valuation Conclusion

| Metric                   | Value          |
| ------------------------ | -------------- |
| Enterprise Value         | ~$612.4B       |
| Equity Value             | ~$635.5B       |
| Intrinsic Value          | ~$145.98/share |
| Market Price (Feb. 2025) | ~$150.68/share |
| Conclusion               | Fairly Valued  |

The analysis suggests ExxonMobil was trading close to its intrinsic value under normalized mid-cycle assumptions.

## Methodology

### Historical Analysis

Financial data from ExxonMobil's Form 10-K filings (2014–2024) was used to analyze:

* Revenue
* EBIT
* NOPAT
* Capital Expenditures
* Depreciation & Depletion
* Net Working Capital
* Free Cash Flow to Firm (FCFF)

### Valuation Framework

* FCFF DCF Model
* 10-Year Forecast Period
* Mid-Cycle Operating Assumptions
* WACC-Based Discounting
* Terminal Value Using Perpetual Growth

### Key Assumptions

| Assumption           | Value  |
| -------------------- | ------ |
| EBIT Margin          | 10.78% |
| Tax Rate             | 25.87% |
| Revenue Growth       | 2.08%  |
| WACC                 | 5.78%  |
| Terminal Growth Rate | 2.00%  |

### Sensitivity Analysis

Valuation outcomes were stress-tested across:

* WACC assumptions
* Terminal growth rates
* Beta estimates
